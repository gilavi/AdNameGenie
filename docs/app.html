Your DMG file is downloading. Here's how to install NameCraft:

📦 Installation

  1. Open the downloaded NameCraft-1.0.0.dmg file
  2. Drag NameCraft into your Applications folder

⚠️ First Launch (macOS Gatekeeper)

macOS may block the app the first time, saying "Apple could not verify…". This is normal for apps outside the App Store.

  1. Right-click the app in Applications → choose Open
  2. Click "Open" in the dialog that appears

OR USE TERMINAL (ONE-TIME FIX)

xattr -dr com.apple.quarantine /Applications/NameCraft.app

Paste this in Terminal after installing. After that it opens normally forever.
